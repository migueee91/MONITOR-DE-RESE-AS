# Monitor de Reseñas - Instrucciones de Deploy

## Variables de entorno necesarias en Render:

| Variable | Valor |
|----------|-------|
| GOOGLE_PLACES_API_KEY | Tu API Key de Google Places |
| GEMINI_API_KEY | Tu API Key de Gemini |
| TELEGRAM_BOT_TOKEN | Token del bot de Telegram |
| TELEGRAM_CHAT_IDS | ID/s separados por coma (ej: 123456,789012) |

## Pasos para subir a Render:
1. Crear cuenta en github.com
2. Subir esta carpeta como repositorio
3. Crear cuenta en render.com
4. Nuevo servicio → conectar repositorio de GitHub
5. Agregar las variables de entorno
6. Deploy!
